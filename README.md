**CS_Mastery_Hub_App** is a premium, high-performance quiz application designed for computer science students, professionals, and trivia enthusiasts. It combines a vast academic database with a sleek, modern dark-themed interface to provide an immersive learning experience.

---

## 🚀 Key Features

- **🏆 30+ Academic Subjects**: Comprehensive quizzes covering everything from *Programming Fundamentals* to *MLOps* and *Generative AI*.
- **⚡ Hybrid Data Engine**: Seamlessly switches between a high-quality local academic database and the **Open Trivia Database (OpenTDB)** for endless variety.
- **🎨 Dark Neon UI**: A custom-built, modern aesthetic featuring **Glassmorphism**, neon indigo accents, and smooth transitions.
- **🔊 Audio Feedback**: Interactive sound effects for correct and incorrect answers to enhance engagement.
- **📊 Local Leaderboard**: Persists your high scores locally, allowing you to track your mastery across different categories.
- **⏱️ Advanced Timer System**: Customizable countdown timers for a pressured, exam-like environment.
- **📱 Fully Responsive**: Optimized for desktops, tablets, and mobile devices.

---

## 🛠️ Technology Stack

- **Frontend Core**: [React.js](https://reactjs.org/)
- **UI Framework**: [Semantic UI React](https://react.semantic-ui.com/)
- **State Management**: React Hooks (useState, useEffect, etc.)
- **Styling**: Vanilla CSS with Glassmorphism and Neon variables
- **Sound Engine**: Custom HTML5 Audio Integration
- **Data Source**: Local JSON DB + [OpenTDB API](https://opentdb.com/)

## ⚙️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Version 16 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/adnanbhutta/CS_Mastery_Hub_App.git
   cd CS_Mastery_Hub_App
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```
